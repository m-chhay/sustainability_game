# Data Center Sustainability Game

## Project Overview

An interactive simulation that allows users to explore how sustainability initiatives impact data center operations, operational costs, energy consumption, and long-term growth.

Designed for technology leaders, sustainability teams, and analysts, the game demonstrates the trade-offs organizations face when balancing profitability with environmental responsibility.

## Features

- Interactive sustainability decision-making simulation
- Dynamic trade-offs between cost, growth, and emissions
- Scenario-based gameplay mechanics
- Real-time score and metric updates
- Responsive desktop-friendly UI

---

## Why I Built This

I created this project to combine my interests in sustainability strategy, enterprise technology, and interactive analytics.

After researching IT sustainability initiatives across industries, I wanted to build a simulation that helps users visualize how operational decisions can affect both business outcomes and environmental impact.

---

## Simulation Logic

The game models how sustainability investments influence:

- Operational cost
- Energy efficiency
- Carbon impact
- Business growth

User decisions dynamically affect performance metrics throughout the simulation experience.
